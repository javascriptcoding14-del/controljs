# If-Else Example 

A small JavaScript project demonstrating `if/else` logic running in the browser.

## Project structure

- [index.html](index.html) — HTML entry file
- [script.js](script.js) — JavaScript with the `if/else` examples

## Prerequisites

- A modern web browser (Chrome, Edge, Firefox)
- VS Code (optional, recommended)
- Node.js or Python (optional, for serving from the console)

## Run using VS Code

1. Open the project folder in VS Code (`File → Open Folder...`).
2. Recommended: install the **Live Server** extension.
3. Open [index.html](index.html), then right-click the editor and choose **Open with Live Server**.
4. The page will open in your browser and reload automatically on save.

If you don't use Live Server, you can open [index.html](index.html) directly in the browser, but some features (like fetching local resources) may require serving over HTTP.


## Edit & develop

- Edit `script.js` in your editor and save. If using Live Server or a local server, reload the page to see changes.
- Keep logic examples small and focused; this project is for learning `if/else` statements.
